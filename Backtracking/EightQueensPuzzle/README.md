Eight Queen Puzzle is about placing n non-attacking queens on an n√ón matrix, for which solutions exist for all natural numbers n with the exception of n = 2 and n = 3. A queen attacks another if:
<ul>
	<li>They are in the same row</li>
	<li>They are in the same column</li>
	<li>They are in the same diagonal</li>
</ul>

<b>Note:</b> This problem is solved using backtracking.
